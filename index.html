Procédure de deploiement

Utilitaires utilisés : github et netifly

  1. Aller sur github et creer le du dépot en suivant ces etapes :
    • Aller à l'interieur de votre projet ex : mon-site
    • git init
    • git add .
    • git commit -m "first commit"
    • git branch -M main
    • git remote add origin https://github.com/digitaladi/nom_du_depot.git
    • git push -u origin main
    • ...
    • et maintenant le code de votre site "mon-site" est dans le repot github nom_du_depot
  2. Aller dans votre compte netlify si vous avez un compte : https://app.netlify.com/teams/digitaladi/overview
    • cliquer sur "Import from Git" pour importer le dépot "nom_du_depot" github
    • cliquer sur "Deploy nom_du_depot"
    • vous avez la possibilté de changer le nom de site
    • définir un nom de domaine
    • securiser l'application avec https
    • ...
    • Maintenant votre site et buildé et déployé en production
